# Branch

- Branch : 나뭇가지의 비유적 표현
- Git Branch는 매우 가볍다.
- 순식간에 Branch를 만들고 Branch 사이를 이동할 수 있다.
- git이 가지고 온 혁신 중 하나는 Branch 기능을 매우 쓸만한 수준까지 만들었다는 것.
- 브랜치 이동 시에는 마스터는 변화 없음.
- 만약 브랜치에서 작업하다가 다시 master로 돌아오면 브랜치에서 작업하는 파일은 안 보인다.(별개의 공간)



- 브랜치 확인

> $ git branch

- 브랜치 만들기

> $ git branch <이름>

- 브랜치 이동

> $ git checkout <이름>
